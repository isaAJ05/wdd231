# WDD 231 - Web Frontend Development

## 📚 Descripción del Curso

Bienvenido a **Web Frontend Development**. Este curso se basa en tu experiencia previa con las tecnologías fundamentales del diseño y desarrollo web, incluyendo HTML, CSS, JavaScript y diseño. El curso se enfoca en la experiencia del usuario, accesibilidad, cumplimiento de estándares, optimización del rendimiento y uso básico de APIs.

Al completar este curso, tendrás un dominio sólido de las tecnologías web fundamentales de HTML, CSS y JavaScript, y estarás preparado para la siguiente secuencia de cursos de aplicaciones.

## 🎯 Objetivos de Aprendizaje

- Dominar las tecnologías fundamentales: HTML, CSS y JavaScript
- Implementar principios de experiencia de usuario (UX)
- Desarrollar sitios web accesibles y que cumplan con los estándares
- Optimizar el rendimiento de aplicaciones web
- Integrar y consumir APIs básicas
- Aplicar mejores prácticas de desarrollo frontend moderno

## 🛠️ Tecnologías Utilizadas

- **HTML5** - Estructura y marcado semántico
- **CSS3** - Estilos, diseño responsivo y animaciones
- **JavaScript (ES6+)** - Interactividad y lógica del lado del cliente
- **APIs** - Integración de servicios externos
- **Git & GitHub** - Control de versiones
