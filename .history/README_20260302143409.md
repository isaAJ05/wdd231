# WDD 231 - Web Frontend Development

## 📚 About This Course

This repository contains my coursework for **Web Frontend Development** at BYU-Idaho. This course builds on my prior experience with HTML, CSS, JavaScript, and design, focusing on user experience, accessibility, compliance, performance optimization, and basic API use.

Through this course, I will develop a firm grasp of core web technologies and prepare myself for the next sequence of application courses.

## 🎯 What I'm Learning

- Mastering core web technologies: HTML, CSS, and JavaScript
- Implementing user experience (UX) principles
- Developing accessible and standards-compliant websites
- Optimizing web application performance
- Integrating and consuming basic APIs
- Applying modern frontend development best practices

## 🛠️ Technologies I'm Using

- **HTML5** - Semantic markup and structure

- **APIs** - External service integration
- **Git & GitHub** - Version control
